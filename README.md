# hichord

Norns port of HiChord (hichord.shop) firmware 2.6.9. Grid interface replicates the physical HiChord hardware — key selector, chord buttons, joystick, waveform select, ADSR, loops, and performance modes.

## Description

HiChord brings the capabilities of the dedicated HiChord hardware instrument to norns. The grid interface faithfully replicates the physical hardware layout with key selector, chord button grid, joystick input, waveform selection, ADSR envelope controls, and loop functionality. All the expressive chord performance modes from the original hardware are available.

A powerful chord instrument with deep harmonic and tonal control.

## Installation

`;install https://github.com/jamminstein/hichord`

## Controls

- **K1**: Toggle page A/B
- **K2**: F2 function
- **K3**: F3 function
- **E1**: Key/browse parameter
- **E2**: Context-dependent control
- **E3**: Context-dependent control

### Grid Interface
The grid mirrors the HiChord hardware layout:
- Left region: Key selector and chord buttons
- Center region: Performance mode controls
- Right region: Waveform, ADSR, and loop controls

## Features

- Complete HiChord firmware 2.6.9 implementation
- Hardware-accurate grid layout
- Key selector with transpose
- Chord button matrix
- Joystick-style pitch/mod input
- Waveform selection
- ADSR envelope controls
- Loop recording and playback
- Multiple performance modes
- MIDI output

## Requirements

- norns
- Grid 16x8
- SuperCollider engine (Engine_HiChord.sc)
